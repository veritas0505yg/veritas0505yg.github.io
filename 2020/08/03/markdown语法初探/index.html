markdown语法初探

发表于 2020-08-03 分类于 markdown

markdown语法初探

一、标题

在想要设置为标题的文字前面加#来表示

一个#是一级标题，二个#是二级标题，以此类推。支持六级标题。标准语法一般在#后跟个空格再写文字

二、字体

  • 加粗

    要加粗的文字左右分别用两个*号包起来

    **我变粗了**

    效果：我变粗了

  • 斜体

    要倾斜的文字左右分别用一个*号包起来

    *我变斜了*

    效果：我变斜了

  • 斜体加粗

    要倾斜和加粗的文字左右分别用三个*号包起来

    ***我变斜加粗了***

    效果： 我变斜加粗了

  • 删除线

    要加删除线的文字左右分别用两个~~号包起来

    ~~这是一句要删除的话~~

    效果：这是一句要删除的话

四、分割线

三个或者三个以上的 - 或者 * 都可以。

五、图片

  • 接着一个普通括号，里面放上图片的网址，最后还可以用引号包住并加上选择性的 ‘title’ 属性的文字。
