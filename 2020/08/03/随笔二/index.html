下阶段感觉后面全是需要填的大坑，先试着理清一下总体路线图和workflow

  • 先说最先开启的深度学习这个坑，原理书你得多少看点吧，这里推荐两本，主要是两位书的作者支持将知识开放出来、以及译者们和社区同仁们的努力。

    • 神经网络与深度学习
    • 动手学深度学习

  • 然后咱得回溯到深度学习的上一级的机器学习吧，这个也得多少看一点吧，这两本书的老师暂时没有开源他们的教材，有点可惜

    • 周志华的《机器学习》
    • 李航的《统计学习方法》

  • 请自觉（p）review概率论与数理统计（尤其数理统计这一块）

  • 机器学习、深度学习读完原理之后还得配合实战书籍和项目进行实操

  • python 自学，又是一个大坑

  • 数字图像处理和机器视觉这一块，感觉坑越来越大了

    • 高敬鹏2020出的两本实战书《机器学习——基于opencv和python的智能图像处理》、《深度学习——卷积神经网络技术与实践》可以看看、
    • 图像处理、分析与机器视觉（清华第四版）和 计算机视觉———特征检测及应用 （机械工业）

  • ENVI、ARCGIS似乎忘了不少，脱产一年多了，感觉还得重新补一下这两者的课
