python第二篇总结笔记

字符串

字符串的本质是字符序列。python字符串是不可变的，无法对原字符串做任何修改，但可以将字符串的一部分复制到新创建的字符串，此外python不支持单字符类型，单字符也被作为一个字符串使用

字符串的编码

python3默认Unicode16位的编码，ASCII码是Unicode的子集
**使用内置函数ord()可以把字符转换成Unicode吗，使用内置chr()可将十进制转换成对应字符**

引号创建字符串

可使用单引号或双引号创建字符串，使用两种引号的好处可以创建本身就包含引号的字符串，而不用使用转义字符

连续三个单引号或三个双引号，可帮助创建多行字符串

空字符串和len函数

python允许空字符串存在，不包含任何字符且长度为0

len()用于计算字符串中有多少字符

转义字符

通过转义字符，实现某种效果

转义字符 描述
(在行尾时) 续行符
\ 反斜杠符号
' 单引号
" 双引号
\b 退格
\n 换行
\t 横向制表符
\r 回车

字符串拼接

原则

  • 两边是字符串，则拼接
  • 两边是数字则加法运算
  • 两边类型不同，抛出异常

字符串成倍复制

不换行打印

前面调用print，会自动打印一个换行符，又是不想自动添加换行符，可通过参数end=“任意字符串”，实现末尾添加任何内容

从控制台获取字符串

使用input从控制台获取键盘输入的内容

str()实现其他数据类型转字符串

str()实现其他数据类型转字符串,当调用print()函数时，解释器自动调用了str()将非字符串的对象转成了字符串

使用[]提取字符

字符串本质是字符序列，可在字符串名后加[]，在[]中指定偏移量

最左侧第一个字符偏移量为0，最右边一个字符为len(str)-1
反向搜索：
最右侧第一个字符偏移量是-1，最左边一个字符是-len(str)

replace()实现字符串替换

有时需要替换字符串，只能通过创建新的字符串实现

slice切片操作

帮助快速提取子字符串。标准格式为：

[起始偏移量start:终止偏移量end:步长:step]

典型操作(三个量皆为正数的情况)如下：

操作和说明 示例 结果
[:]提取整个字符串 “abcdef”[:] “abcdef”
[start:]从start索引开始到结尾 “abcdef”[2:] “cdef”
[:end]从头开始到end-1 “abcdef”[:2] “ab”
[start:end]从start开始到end-1 “abcdef”[2:4] “cd”
[start:end:step]从start开始到end-1 “abcdef”[1:5:2] “bd”

其他操作（三个量为负数时）

示例 说明 结果
“abcdefgh”[-3:] 倒数三个 “fgh”
“abcdefgh”[-8:-3] 倒数第八个到倒数第三个(左包含右不包含) “abcde”
“abcdefgh”[::-1] 步长为负，从右到左反向提取 “hgfedcba”

split()分割和join()合并

split()基于指定分割符。将字符串分割成多个字符子串（存储到例表中）。如果不指定分割符，则默认使用空白字符（换行符/空格/制表符）

join()正好与split作用相反，用于将一系列子字符串连接起来

使用拼接字符+，会生成新的字符串对象，因此不推荐使用+而是使用join函数

字符串比较和同一性

直接可以使用==，！=对字符串进行比较，是否有相同的字符

使用is/not is，判断两个对象是否同一个对象，比较的是二者的地址，即id(obj1)是否等于id(obj2)

成员操作符

in/not in判断字符是否存在于字符串中

字符串驻留机制和字符串机制

仅保存一份相且不可变字符串的方法，不同的值被存放在字符串驻留池中。python对于符合标识符规则的字符串（仅包含下划线(_)、字母和数字）

去除首尾信息

可以通过strip()去除字符串首尾指定信息。通过lstrip()去除字符串左边指定的信息，rstrip()去除字符串右边指定信息

大小写转化

示例 说明
a.capitalize() 产生新的字符串，首字母大写
a.title() 产生新的字符串，每个单词的首个字母都大写
a.upper() 产生新的字符串，所有字符全都转化成大写
a.lower() 产生新的字符串，所有字符全都转化成小写
a.swapcase() 产生新的，所有字母大小写互换

字符串的格式化

format()基本用法
str.format()通过{}和:代替以前的%
format函数可以接受无限个参数，位置可以不按顺序

填充与对齐

填充与对齐常一齐使用
^、<、>分别是居中、左对齐、右对齐，后面带宽度
:号后面带填充的字符，只能是一个字符，不指定的话默认是用空格填充

数字格式化

数字 格式 输出 描述
3.1415926 {:.2f} 3.14 保留小数点后两位
3.1415926 {:+.2f} 3.14 保留小数点后两位
3.1415926 {:.0f} 3 不带小数
5 {:0>2d} 05 数字补零（右对齐，宽为2）
1000000 {:,} 1,000,000 逢千分割
0.25 {:.2%} 25.00% 百分比格式

可变字符串

有时我们需要修改原字符串对象，可以使用io.StringIO对象或array对象
